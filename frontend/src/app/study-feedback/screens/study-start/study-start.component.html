Welcome to our evaluation study!

This study aims for understanding, how potential users interact with this prototype and whether it can help them give better feedback in typical scenarios of supervising software engineering student projects.

You will be presented a sequence of seven small tasks, referring to anonymized, real-world projects. Please solve these tasks using the information offered on this platform. You may exit the study at any given time, but please take notice, that a complete run of this study may produce the most reliable results.

Keep in mind, that you can minimize and maximize this window at any time of the study, in order to view the tool in fullscreen.

We are logging relevant interaction events for later evaluation:

timeline

navigation paths and dwell times

 list_alt

evaluation system interaction

 dns

system events

 mouse

click and hover events

 schedule

event timestamps

If you have any doubts later on, you may request deletion of your generated data. Simply send a mail to Felix Riedl

By clicking the BEGIN-button, you agree on the logging and submission of the previously mentioned events and start the study.
